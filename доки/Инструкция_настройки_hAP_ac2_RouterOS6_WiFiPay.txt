ИНСТРУКЦИЯ: БЫСТРАЯ НАСТРОЙКА WI-FI НА MIKROTIK (HAP AC2)
-------------------------------------------------------
Проблема: Новые драйверы (WiFi/Wave2) часто блокируются на 
hAP ac2 из-за ошибки частоты процессора.
Решение: Использование стабильного пакета "Wireless".

ШАГ 1: УСТАНОВКА ПРАВИЛЬНОГО ДРАЙВЕРА
1. Зайти в System -> Packages. 
2. Если видишь "wifi-qcom-ac" -> нажми Uninstall.
3. Скачай с сайта Mikrotik пакет "wireless-(твоя версия)-arm.npk".
4. Закинь файл в корень меню Files (НЕ в папку flash!).
5. Сделай System -> Reboot.
6. После перезагрузки слева должно появиться меню "Wireless".

ШАГ 2: ОТКРЫТАЯ СЕТЬ (БЕЗ ПАРОЛЯ)
1. Wireless -> Security Profiles.
2. Открой профиль "default".
3. Mode: поставь "none".
4. Нажми Apply и OK.

ШАГ 3: НАСТРОЙКА И ВКЛЮЧЕНИЕ АНТЕНН
1. Wireless -> WiFi Interfaces -> дважды кликни на wlan1 (5GHz) или wlan2 (2.4GHz).
2. Вкладка Wireless:
   - Mode: ap bridge
   - Band: 5GHz-A/N/AC (или 2GHz-G/N для wlan2)
   - SSID: Название твоей сети (например, Wifi_pay-16)
   - Security Profile: default
3. Нажми Apply.
4. Нажми СИНЮЮ ГАЛОЧКУ (Enable) вверху окна.

ШАГ 4: ИНТЕРНЕТ (BRIDGE)
1. Bridge -> Ports -> синий "+".
2. Interface: выбери свой wlan.
3. Bridge: выбери основной bridge.
4. Нажми OK.

ШАГ 5: ХОТСПОТ НА 3 МИНУТЫ (TRIAL)
1. IP -> Hotspot -> Кнопка "Hotspot Setup".
2. Interface: bridge. Далее везде "Next" до конца.
3. В окне Hotspot перейди в "Server Profiles".
4. Открой созданный профиль -> вкладка Login.
5. Поставь галочку "Trial".
6. Trial Uptime Limit: 00:03:00 (твои 3 минуты).
7. Trial Uptime Reset: 1d 00:00:00 (доступ раз в сутки).
8. Нажми OK.

ГОТОВО! Сеть работает, ограничение в 3 минуты активно.
